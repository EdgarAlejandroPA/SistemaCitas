## Instalación y configuración
1. Clona el repositorio: `git clone https://github.com/tu-usuario/tu-repositorio.git`
2. Asegúrate de tener instalado JDK 11.
3. Compila y genera el FAT JAR: `mvn clean package` o `gradle build`.

## Uso del programa
Ejecuta el programa con: `java -jar nombre-del-archivo.jar`.
